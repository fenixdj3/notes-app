# Notes App

A clean and responsive notes application built with HTML, CSS, and vanilla JavaScript. Users can create, edit, delete, pin, and search notes. Notes are saved in the browser with localStorage.

This is a beginner frontend practice project created for a personal portfolio.

## Demo

[https://fenixdj3.github.io/notes-app](https://fenixdj3.github.io/notes-app)

## Features

- Create notes with a title and text
- Validate empty form fields
- Edit existing notes
- Delete notes with a confirmation message
- Pin and unpin important notes
- Search notes by title or text
- Save notes with localStorage
- Show an empty state when there are no notes
- Responsive layout for desktop and mobile screens

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Browser localStorage

## How to Run the Project Locally

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in a web browser.

No installation, build tools, or external libraries are required.
